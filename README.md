# Advisory Match Platform

Enhanced financial advisor matching platform with 24 enterprise features.

## Features

- ✅ ML-inspired matching algorithm
- ✅ Dark mode (3 themes)
- ✅ Mobile optimized
- ✅ Offline support
- ✅ Command palette (⌘K)
- ✅ Advanced search & filters
- ✅ Toast notifications
- ✅ Keyboard shortcuts
- ✅ And 16 more enterprise features!

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Deploy

## Environment Variables

Create a `.env.local` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
